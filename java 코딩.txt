자바 코드 작성 및 실행
1. 소스코드 작성 : 메모장, Sublime text 등의 편집기를 사용하여 *.java 문서를 만든다
=> Hello.java 파일 생성했다

2. 컴파일(Compile) : 작성된 소스코드를 기계어 코드로 번역하는 것 *.class 파일이 만들어 진다
=> Hello.class 파일 생성되었다.

3. 실행(Execution) : 컴파일된 기계어 코드를 컴퓨터에게 주입하여 연산을 수행하는 일

소스코드 : 메모장, sublime text 등으로 만든 파일
이진파일 : 컴파일된 기계어 코드가 저장된 파일

컴파일 : 소스코드 => 번역 => 기계어 코드
실행 : 기계어 코드 => 운영체제에게 전달하여 => 결과를 내도록 하는 것

컴파일 : javac 소스코드.java => 소스코드.class
실행 : java 소스코드 => 결과 확인
