# 연산
* 연산은 산술연산과 논리연산이 있다

## 산술연산(Arithmetic)
* 사칙연산 처럼 어떤 결과가 확실하게 도출되는 연산

## 논리연산(Logic)
* 어떤 대상끼리 관계를 비교하여 다른 결과를 찾아내는 연산

## 컴퓨터의 CPU
* ALU(연산장치), Controller(제어장치)
* ALU(Arithmetic Logic Unit)
